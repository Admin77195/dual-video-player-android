========================================
  📱 如何从 GitHub 下载打包的 APK
========================================

当前状态：您在 GitHub Actions 的初始页面
目标：找到并下载编译好的 APK 文件

========================================
  🎯 方法1：查看已完成的构建（推荐）
========================================

【步骤1】进入 Actions 页面
-----------------------------------
在您的 GitHub 仓库页面：
1. 点击顶部的 "Actions" 标签（您现在的位置）
2. 应该会看到工作流运行列表

【步骤2】查看构建记录
-----------------------------------
在 Actions 页面：
1. 左侧应该显示 "Build Android APK" 工作流
2. 中间显示运行记录列表
3. 找到最新的一条记录（通常在最上面）
4. 查看状态：
   ✅ 绿色勾号 = 构建成功
   🔄 黄色圆圈 = 正在构建
   ❌ 红色叉号 = 构建失败

【步骤3】下载 APK
-----------------------------------
如果构建成功（绿色勾号）：
1. 点击该运行记录
2. 滚动到页面底部
3. 找到 "Artifacts" 部分
4. 点击 "android-apk" 下载 ZIP 文件
5. 解压 ZIP 得到 APK 文件

【APK 文件名】
DualVideoPlayer-1.0-debug.apk


========================================
  🎯 方法2：手动触发构建
========================================

如果没有看到任何构建记录，需要手动触发：

【步骤1】进入工作流页面
-----------------------------------
1. 点击 "Actions" 标签
2. 左侧点击 "Build Android APK"

【步骤2】运行工作流
-----------------------------------
1. 右侧点击 "Run workflow" 按钮（绿色）
2. 在弹出的对话框中：
   - Branch: 选择 "main" 或 "master"
   - 点击绿色的 "Run workflow" 按钮

【步骤3】等待构建完成
-----------------------------------
1. 构建时间：约 20-30 分钟
2. 可以点击运行记录查看实时日志
3. 构建完成后按方法1下载 APK


========================================
  🎯 方法3：检查 Releases
========================================

如果配置了自动发布，也可以在 Releases 查看：

1. 点击仓库右侧的 "Releases"
2. 查看最新的发布版本
3. 下载 Assets 中的 APK 文件

注意：当前配置没有自动发布到 Releases
      需要手动从 Artifacts 下载


========================================
  ❓ 常见问题
========================================

【问题1】看不到 Actions 标签？
-----------------------------------
原因：Actions 可能未启用
解决：
1. 进入仓库 Settings
2. 左侧点击 Actions → General
3. 确保 Actions 已启用

【问题2】构建失败（红色叉号）？
-----------------------------------
原因：可能是配置问题或依赖问题
解决：
1. 点击失败的运行记录
2. 查看错误日志
3. 根据错误信息修复

【问题3】找不到 Artifacts？
-----------------------------------
原因：构建可能还在进行中
解决：
1. 等待构建完成（绿色勾号）
2. 刷新页面
3. 滚动到页面底部查看

【问题4】下载的是 ZIP 文件？
-----------------------------------
这是正常的！
1. 解压 ZIP 文件
2. 里面有 APK 文件
3. 将 APK 传输到手机安装


========================================
  📱 安装 APK 到手机
========================================

【方法1】USB 传输
-----------------------------------
1. 用 USB 线连接手机和电脑
2. 将 APK 复制到手机
3. 在手机上找到 APK 文件
4. 点击安装（需要允许安装未知来源应用）

【方法2】云盘传输
-----------------------------------
1. 上传 APK 到云盘（百度网盘、微云等）
2. 手机上下载 APK
3. 点击安装

【方法3】扫码传输
-----------------------------------
1. 使用文件传输工具（如：快牙、茄子快传）
2. 扫码传输 APK
3. 点击安装
